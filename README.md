# umair-as.github.io/blog

Personal engineering blog. Writing about embedded Linux, Yocto, Zephyr, wireless protocols, and IoT infrastructure — from real projects, not toy examples.

**Live:** https://umair-as.github.io/blog/

## Stack

- [Hugo](https://gohugo.io/) (extended) — static site generator
- [PaperMod](https://github.com/adityatelange/hugo-PaperMod) — theme
- GitHub Pages — hosting
- GitHub Actions — build and deploy on push to `main`

Embedded Linux · Yocto/OpenEmbedded · systemd hardening · OpenThread · Zephyr RTOS · RAUC OTA · IoT gateway infrastructure
